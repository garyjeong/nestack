# Nestack Color Concept (v1)

## 1) 컬러 컨셉 요약 (No Blue)
- **Primary (성장/진행)**: Emerald 계열 — 미션/자산 “성장”의 의미를 UI 전반에 일관 적용
- **Accent (커플 감성/강조)**: Rose 계열 — 뱃지/하이라이트/핵심 CTA 포인트
- **Secondary (중립/관리 UI)**: Stone 계열 — 서브 버튼/보조 UI/어드민의 안정감
- **Neutral (기본 톤)**: Gray 계열 — 배경/텍스트/보더

## 3) 상태 컬러 (Status)
- **Success**: `#10B981` (또는 버튼/강조 시 `#059669`)
- **Warning**: `#F59E0B`
- **Error**: `#EF4444`
- **Info**: `#84CC16` (Blue 계열 미사용 원칙 유지)

## 4) 사용 규칙 (권장)
- **Primary**: 메인 네비/핵심 CTA/진행률/핵심 지표(자산, 미션 진행)
- **Accent**: 뱃지, “축하/하이라이트”, 2차 강조 CTA (과사용 금지)
- **Secondary**: 서브 버튼, 필터/정렬, 어드민 UI의 주요 액션(Primary와 역할 분리)
- **Neutral**: 배경/카드/디바이더/본문 텍스트
